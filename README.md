### Ejercicios XQUERY

> [!NOTE]
> para los datos se utiliza _**peces.xml**_

**app1_text.xq** → salida1.html
```
basex -i peces.xml -o salida1.txt -Q app1_text.xq ; type salida1.txt ;
```

**app2_xml.xq** → salida2.xml
```
basex -i peces.xml -o salida2.xml -Q app2_xml.xq ; type salida2.xml ;
```

**app3_html.xq** → salida3.html
```
basex -i peces.xml -o salida3.html -Q app3_html.xq ; type salida3.html ;
```

**app4_css.xq** → salida4.html
```
basex -i peces.xml -o salida4.html -Q app4_css.xq ; type salida4.html ;
```
